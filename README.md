# Task: Course Hub – Online Course Management System

This task involves the design and development of a web-based application named Course Hub. The objective of this project is to create a simple and user-friendly platform where users can explore, select, and manage different online courses efficiently. The system is designed to help beginners understand course selection and management in a structured digital environment.
